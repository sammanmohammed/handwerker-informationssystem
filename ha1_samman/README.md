# 🛠️ Handwerker-Informationssystem

Dieses Projekt ist im Rahmen des Moduls **Objektorientierte Programmierung III** entstanden.

## 📌 Beschreibung
Entwicklung einer Java-Anwendung zur Verwaltung von Kunden, Aufträgen und Ressourcen.  
Die Anwendung bietet eine grafische Benutzeroberfläche und speichert Daten persistent in einer Datenbank.

## 🚀 Funktionen
- Verwaltung von Kunden
- Erstellung und Bearbeitung von Aufträgen
- Ressourcenmanagement
- Speicherung und Abruf von Daten aus der Datenbank

## 🏗️ Technologien
- Java
- JavaFX (GUI)
- JPA (Datenbankzugriff)
- Maven (Build-Tool)
- SQL / H2 / Oracle

## 🧩 Architektur
Die Anwendung folgt einer mehrschichtigen Architektur:
- Präsentationsschicht (JavaFX)
- Anwendungskern (Geschäftslogik)
- Persistenzschicht (JPA)
